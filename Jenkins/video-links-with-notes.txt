docker run -p 8080:8080 -p 50000:50000 -d -v jenkins_home:/var/jenkins_home jenkins/jenkins

-p 8080 = by default runs on this port
-p 50000 = master/slave communication
-d = run container in the background
-v = persistent data of Jenkins in a volume, with folder name and path.

docker logs <container_ID>
(scroll until you find the text with 'the following password to proceed to installation' to retrieve the initialization password)

Create your first username and password for Jenkins.

Types of Jenkins projects:
Freestyle - simple, single tasks, run tests.
Pipeline - whole delivery cycle, test, build, for a single branch.
Multibranch pipeline - like pipeline for multiple branches.
