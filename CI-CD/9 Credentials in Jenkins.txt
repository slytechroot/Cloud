Credentials in Jenkins <--------------

Username with passwords are the most common.

Credentials Plugin installed.
- store and manage credentials centrally.

Global (Jenkins, Nodes, items, all child items, etc.)
- available Everywhere!

vs 

System (Jenkins and nodes ONLY)
- only available on Jenkins server, NOT for Jenkins jobs.

Credentials types:
Github app
SSH username with private key
Secret file
Secret text
Certificate

###########

- new types may be released based on plugins!!!!!

Credentials might not be visible by others!

The Multibranch type has the Credentials options, the other types don't have this!
