DEMO
Deploy Web Application on EC2 instance:
- Create an EC2 instance on AWS
- Connect to EC2 instance with SSH
- Install Docker on remote EC2 instance
- Run Docker Container (docker login, pull, run) from Private Repo.

Tags are Metadata that can help us differentiante between images, their purpose, etc.

Configure Security Groups - or firewall rules on Subnet or Instance level.

Create the EC2 instance!!!!

Login over SSH, but before.
Save the private.pem file to local machine.
Give it only read permissions:
chmod 400 .ssh/private.pem

ssh -i ~/.ssh/docker-server.pem ec2-user@<public.ip>

Update your package manager tool!
For up-to-date repositories!

sudo yum update

#install Docker
sudo yum install docker

Start the Docker daemon:
sudo service docker start

sudo docker run

, but we want to add the ec2-user to the Docker group
sudo usermod -aG docker $USER
